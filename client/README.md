# SynthMeet Client

React + Vite frontend for SynthMeet.

## Responsibilities

- Audio upload and in-browser recording UI
- Meeting history, search, and export screens
- Integration with the API through `src/lib/api.js`

## Development

```bash
npm install
npm run dev
```

## Environment

Create `client/.env` from `client/.env.example` when you need to point the UI at a non-default API origin.
